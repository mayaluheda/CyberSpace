PROJECT TITLE: CyberSpace
PURPOSE OF PROJECT: An application that visits web pages and counts the lines in each one's html code. It reports this information to the user. Program output:
1. its url (like "http://cs.wellesley.edu/~cs230/index.html"),
2. the number of lines in its html code
3. its actual html code (only the first 30 characters.)
4. the largest web page (largest number of html code.)

USER INSTRUCTIONS: input a working URL to receive relevant info
